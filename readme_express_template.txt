	Installing EXPRESS in a CLI
	***************************



	1) Create a Working Directory to hold your application (make that your working directory)
  	   and go to it (into it):

$  mkdir  myapp_working_directory

$  cd     myapp_working_directory



	2) Initialize your application (working directory) with NPM (Node Package Manager)
	   Creates a package.json (configuration file)
	   (inside your working directory)

$  npm  init  -y



	3) Install Express in your working directory

$  npm  install  express



	4) Create a JavaScript file:	server.js
	   Note: The Express generator creates a scaffolding with sub-directories & files.
                     *****************           ***********

$  touch  server.js



	5) Start VS-CODE

$  code .
